# production_data_shengchan 目录说明

这个目录用于本地放置真实生产业务数据。它和 `examples/` 的区别是：这里可以是工作用真实数据，`examples/` 应该是可提交、可分享的脱敏样例。

## 使用建议

| 规则 | 说明 |
|---|---|
| 不提交真实数据 | 生产任务、客户信息、价格、内部排期等都应留在本地 |
| 保持文件名可读 | 建议用日期、来源、用途命名，方便回溯 |
| 样例另放 | 需要共享格式时，复制脱敏版本到 `examples/` |

如果某个生产数据文件被脚本固定读取，建议在对应脚本 README 或脚本配置区注明路径。
