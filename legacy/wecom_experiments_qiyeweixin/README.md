# 企业微信实验脚本

这个目录保存企业微信相关实验脚本，主要用于图形、Excel、甘特图或框架生成方案验证。

| 文件 | 功能 |
|---|---|
| `L03_wecom_gantt_experiment_qiyeweixin.py` | 企业微信甘特图相关实验 |
| `L04_wecom_excel_experiment_qiyeweixin.py` | 企业微信数据到 Excel 的实验 |
| `L05A_wecom_frame_experiment_qiyeweixin.py` | 框架生成实验 |
| `L05_wecom_shape_experiment_qiyeweixin.py` | Excel 形状或排产卡片实验 |

这些脚本用于参考和验证思路，不是当前推荐入口。
