# compat_wrappers_archive 目录说明

这个目录保存旧版脚本、中文旧文件名和兼容包装文件。它的价值是帮助追溯迁移前的使用方式。

## 主要内容

| 类型 | 说明 |
|---|---|
| 编号脚本 | 旧版制造计算流程，如提取、矩阵、成本、利润 |
| 兼容包装 | 指向新入口或保留旧文件名的包装脚本 |
| 平台旧目录 | `企微re/`、`飞书/` 等旧实验目录 |
| 杂项脚本 | PPT、色差、视频裁剪等历史工具 |

## 子目录文档

- [企微re 历史目录](企微re/README.md)
- [飞书历史目录](飞书/README.md)

## 使用建议

新工作优先使用：

- `apps/`：平台同步和排产主入口。
- `tools/`：独立工具。
- `peifang_core/`：共享核心逻辑。
