# 杂项工具

这个目录保存临时或独立用途的工具脚本。

| 文件 | 功能 |
|---|---|
| `M10_message_sender_main.py` | 读取企业微信 webhook 等配置，向指定群机器人发送本地消息 |
| `M20_haier_ppt_gongju.py` | 处理海尔相关演示文稿生成或整理任务 |
| `M30_color_standard_juxian.py` | 整理或具现颜色标准资料，辅助生产沟通 |
| `M40_video_cut_jianji.py` | 执行本地视频剪辑或裁切相关处理 |
| `M50_text_notes_tici.txt` | 文本题词或备注资料 |

## 使用建议

这里的脚本彼此关联较弱，运行前先看脚本顶部说明和配置区。若某个工具变成长期流程的一部分，再考虑迁移到 `apps/` 或 `peifang_core/`。
