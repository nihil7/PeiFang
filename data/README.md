# data 目录说明

`data/` 用于保存同步脚本生成的本地缓存、状态文件和中间数据。这里通常是运行产物，不应提交真实业务数据。

## 常见内容

| 类型 | 来源 | 用途 |
|---|---|---|
| 字段缓存 | 企业微信或飞书同步脚本 | 保存表字段结构，便于后续解析和核对 |
| 记录缓存 | 企业微信或飞书同步脚本 | 保存原始记录、增量记录或合并后的记录 |
| 同步状态 | 同步脚本 | 记录上次同步时间、校验时间或分页状态 |

## 提交建议

- 真实数据不要提交。
- 如果需要给别人演示目录结构，用 `README.sample.md` 或 `examples/` 放脱敏样例。
- 排查同步问题时，可以临时查看这里的 JSON 文件确认字段和值结构。
